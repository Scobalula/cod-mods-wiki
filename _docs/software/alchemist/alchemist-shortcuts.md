# Shortcuts

* CTRL+S: Saves current project
* CTRL+SHIFT+S: Saves current project as a new file
* CTRL+C: Copy selected animations
* CTRL+V: Save selected animations
* CTRL+Q: Attempt best-fit columns to screen
